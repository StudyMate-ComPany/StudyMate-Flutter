## 🎯 주요 기능

앱이 제공하는 핵심 기능:
1. **로그인/회원가입**: 이메일 인증 기반
2. **스터디 그룹**: 생성, 참여, 관리
3. **일정 관리**: 캘린더 뷰, 알림
4. **게시판**: 글 작성, 댓글, 좋아요
5. **학습 추적**: 타이머, 통계
6. **과제 관리**: 생성, 제출, 평가

## 🛠️ 사용 가능한 명령어

Flutter run 실행 중 사용 가능한 키보드 명령:
- `r`: Hot reload (빠른 새로고침) 🔥
- `R`: Hot restart (앱 재시작)
- `h`: 모든 명령어 보기
- `d`: 디버거 분리 (앱은 계속 실행)
- `c`: 화면 지우기
- `q`: 앱 종료
